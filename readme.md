
run app.js as a regular node program to import transactions
needs to import a CSV file with the following fields:
Date,No,Description,Debit,Credit

running the site
make sure to run the docker container named competent_curran.  this is the postgres sql db.
run nodemon from the web-scraping folder.  it runs the index.js file
open browser and go to localhost:3000 (this is the default home)
click the view transactions link to get into the main site pages
